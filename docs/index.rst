Introduction
============

Welcome to the documentation of STM32 driver for BMP180. This driver supports I2C interface.

Source code is available at `Github <https://github.com/ceyhunsen/bmp180-stm32-driver>`_.

Units
^^^^^

* Temperature: Celsius
* Pressure: Pascal
* Altitude: Meter
